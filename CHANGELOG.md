# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial public release
- Offline-first RSS reader with IndexedDB storage
- PWA support for installable web app
- OPML import/export functionality
- Dark/light mode with FT Origami O3 design tokens
- Three-panel layout inspired by FT.com
- Auto-sync with background fetch
- Search functionality with tokenization

### Features
- Feed subscription and management
- Article reading with full-text preservation
- Automatic feed refresh with configurable interval
- Auto-archive strategy (top 50 unread, rest archived)
- Responsive design for desktop and tablet

### Initial Release
- Project foundation with SvelteKit 2
- Dexie.js database setup
- Basic RSS feed parsing
